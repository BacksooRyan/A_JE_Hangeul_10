# 아재한글 1.0 (A_JE_Hangeul_10)

**아두이노 우노(Arduino Uno)**에서도 사용할 수 있도록 만든 **조합형 한글 폰트 라이브러리**입니다.  
영문/숫자 출력만 가능했던 기존 임베디드 환경에서도 **한글을 초소형 메모리로 표시**할 수 있도록 제작했습니다.

## 특징
- **조합형 한글 엔진**: 초성, 중성, 종성을 조합하여 실시간 글자 출력
- **초경량 메모리 구조**: 우노(2KB SRAM)에서도 동작 가능
- **범용 디스플레이 지원**: 128x64 흑백 OLED/GLCD 기준 테스트
- **실험적 프로젝트**: 교육/연습용, 한글 출력 원리 연구에 적합

## 한계점
- 전체 화면 갱신 시 **깜빡임 현상** 발생 가능
- 메모리 제한으로 인해 **더블버퍼링 불가**  
  (대안: **부분 업데이트 방식** 사용 권장)

## 개선 아이디어
- 바뀐 영역만 다시 그리는 **Partial Update** 적용
- **자주 쓰는 글자 캐싱**으로 속도 개선

}
